# Night Runner

A fast-paced mobile arcade game where you guide a glowing character through a neon cityscape at night.

## Features

- **30 Levels** across 3 themed worlds
- **5 Unlockable Characters** with unique visual effects
- **Mobile-First Design** - Optimized for touch controls
- **Level-Based Progression** - No infinite grinding
- **Star Collection System** - Earn stars to unlock characters
- **Juice & Polish** - Satisfying particles, screen shake, and sound effects
- **Neon Synthwave Aesthetic** - Cyberpunk night cityscape

## How to Play

### Mobile
- **Tap screen:** Small jump
- **Hold screen:** Higher jump

### Desktop
- **Spacebar:** Jump
- **Hold Spacebar:** Higher jump

## Progression

- Complete levels to unlock the next one
- Collect stars to unlock new characters
- 30 total levels across 3 worlds:
  - **Neon City** (Levels 1-10)
  - **Cyber Highway** (Levels 11-20)
  - **Space District** (Levels 21-30)

## Characters

1. **Neon Runner** (Default) - Blue glow
2. **Cyber Ninja** (100 stars) - Purple glow, leaves trail
3. **Star Chaser** (250 stars) - Gold glow, magnet effect
4. **Shadow Dash** (500 stars) - Red glow, speed boost
5. **Galaxy Glider** (750 stars) - Rainbow glow, double jump

## Tech Stack

- Vanilla HTML5 Canvas
- Pure JavaScript (no frameworks)
- Web Audio API for sound
- LocalStorage for save data

## Development

Built following proper game design principles:
- Research phase completed before coding
- Mobile-first design and testing
- Level-based progression (not infinite)
- Heavy emphasis on juice and polish
- Clear visual hierarchy and feedback
